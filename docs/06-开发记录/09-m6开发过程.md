### 5.4 已知限制

| 限制 | 说明 | 影响 |
| --- | --- | --- |
| 单进程写入 | 仅支持一个应用进程操作数据文件 | 不支持多实例部署 |
| 无浏览器验证 | M5-05 和 EXIT-05 的 Chrome/Edge 验证未在实际浏览器中执行 | 需在实际环境中补充验证 |
| JSON 文件存储 | 单文件存储，不支持并发写入和高并发 | MVP 小规模演示足够 |
| 无性能指标 | 不设硬性性能阈值 | 小规模数据下流畅使用即可 |
